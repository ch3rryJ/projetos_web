Listas não ordenadas

A tag ul (unordered list - lista não ordenada) representa uma lista de itens sem ordem rígida, por exemplo ingredientes de uma receita.

A tag li (list itens - itens da lista) representa um item que faz parte de uma lista.

exemplo de lista

  • Primeiro item
  • Segundo item
  • Terceiro item
    • subitem

Listas ordenadas

A tag ol (ordered list - lista ordenada) representa uma lista de itens numerados sequencialmente.

Exemplo lista oredenada

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Atribultos

reversed (ordem reversa)
Exibe os itens da lista ordenada na ordem reversa

  1. item mais importante
  2. item menos importante

start (inicio)
especifica o valor inicial para numeração da lista.

  1. item 5
  2. item 6

type (tipo)
indica o tipo de numeração:
a - indica letras minusculas A - indica letras maiusculas i - indica algarismos romanos minusculos I - indica algarismos romanos maiusculos

  1. primeiro item
  2. segundo item
